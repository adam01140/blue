✅ New Feature Added: Right-Click Option Type Change

You can now right-click on any option node to change its type:

  • Regular Option - Simple text option
  • Image Option - Option with image display
  • Amount Option - Option with amount input field

How to Test:

  1. Open the main flowchart application
  2. Drag an "Options Node" from the toolbar to the canvas
  3. Right-click on the option node
  4. Click "Change Option Type »"
  5. Select one of the three option types from the submenu
  6. Observe how the node changes appearance and behavior
Note: This feature works exactly like the existing question node type change feature, but specifically for option nodes. The context menu automatically detects when you right-click on an option node and shows the appropriate submenu.

Technical Implementation Details:

  • Added new HTML submenu for option types
  • Created setOptionType() function to handle option type changes
  • Modified context menu handler to show option submenu for option nodes
  • Added event handlers for the three option type buttons
  • Updated menu text to indicate "Change Option Type" for option nodes
